Model Summary

  • All model code is located within the Assessment 1 Repository (linked above), along with the associated files for the README, Licensing and Documentation.
  • This animated agent-based model uses Spyder (Anaconda 3) Python 3.7.
  • The model code interacts with an agent framework code and an environment to produce coordinates on a 'raster' grid. The coordinates for the 'Sheep' variables are produced randomly and the 'Foxes' are plotted from an online web-derived set of data points.
  • The number of Sheep and Foxes is determined by a GUI slider. Between 50 to 100 Sheep can be chosen and between 5 to 15 Foxes can be chosen.
  • Both the ‘Foxes’ and ‘Sheep’ variables are defined as ‘Agents’ at distinctly defined separate times within the model. The 'Sheep' in the model- move, 'eat' the environment (grass), share food with their nearest neighbour and a selection of the 'Sheep' are unfortunately subsequently 'killed' by the 'Foxes' when the 'Foxes' get sufficiently close.
  • Once either the maximum number of iterations is met or the code is manually closed via the menu bar or the Foxes food storage capacity is reached, the model stops.
  • The code uses core Python language with an object-oriented approach, continuous integration and web-scraping.
